# Weather-Prediction-Using-Machine-Learning


## Overview
This project predicts rainfall using machine learning algorithms applied to the Seattle Weather Dataset. The study compares six supervised learning models to evaluate their performance in rainfall prediction.

## Algorithms Used
- Random Forest
- Logistic Regression
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)
- Decision Tree
- Naïve Bayes

## Dataset
Seattle Weather Dataset containing:
- Precipitation
- Maximum Temperature
- Minimum Temperature
- Wind Speed
- Weather Condition

Target Variable:
Rain = 1, No Rain = 0

## Data Preprocessing
- Missing value handling
- Feature scaling using StandardScaler
- Dimensionality reduction using PCA
- Train-test split (70% training / 30% testing)

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC
- Confusion Matrix

## Results
Random Forest achieved the highest performance with:
- Accuracy ≈ 0.86
- ROC-AUC ≈ 0.93
